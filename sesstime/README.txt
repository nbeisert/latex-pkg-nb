sesstime is a LaTeX2e package to add timing marks to lecture notes
in order to help managing the time available for presenting a given 
section of the document. It also provides tools to record and
estimate the progress throughout the course.

The package consists of the files

    README.txt           readme file
    sesstime.ins         installation file
    sesstime.dtx         source file
    sesstime.sty         package file
    sesstime-samp.tex    sample file
    sesstime-samp-3.tex  sample include file
    sesstime-samp-4.tex  sample include file
    sesstime.pdf         documentation

The distribution consists of the files
README.txt, sesstime.ins and sesstime.dtx.

Run "latex sesstime.ins" to create the package sesstime.sty
and further documentation files.

Copy the file sesstime.sty to an appropriate directory
of your LaTeX distribution, e.g. texmf-root/tex/latex/sesstime.
Alternatively, you may copy sesstime.sty to your project directory.

Run "pdflatex sesstime.dtx" to compile the documentation sesstime.pdf.
